
# 美丽乡村专题网站

> 基于vue的网站

## 技术栈

- 脚手架：vue-cli
- 前端： vue ，vue-router ，vuex
- 打包工具： webpack
- css依赖： stylus

## 项目建立（详情看package.json）

``` bash
# 安装依赖
npm install

# 运行 （访问localhost:8080）
npm run dev

# 打包
npm run build
```
